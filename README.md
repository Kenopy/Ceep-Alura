# Ceep - Gerenciador de Notas e Tarefas

O **Ceep** é uma aplicação prática desenvolvida durante um curso basico de JavaScript focado em manipulação do DOM (Document Object Model). O objetivo principal é oferecer uma interface simples e funcional para gerenciar tarefas diárias.

---

## Funcionalidades (CRUD)

Esta aplicação permite que o usuário realize as operações básicas de um gerenciamento de dados:
* **Create**: Adição de novas tarefas através de um campo de input.
* **Read**: Visualização das tarefas em uma lista vertical organizada.
* **Update**: Marcação de tarefas como concluídas (interação visual).
* **Delete**: Remoção de tarefas da lista de forma individual.

---

## Interface Visual

A interface foi projetada para ser minimalista e eficiente, utilizando uma paleta de cores escuras para reduzir a fadiga visual e ícones intuitivos para as ações de concluir e deletar.

## Tecnologias Utilizadas

* **HTML5**: Estruturação semântica da aplicação.
* **CSS3**: Estilização moderna com foco em Flexbox e transições suaves.
* **JavaScript (ES6+)**: Lógica de programação, criação dinâmica de elementos e manipulação de eventos do DOM.
